# 宝宝湾h5

## Project setup 进行项目安装
```
npm install
```

### Compiles and hot-reloads for development 编译和热重新加载以进行本地开发
```
npm run serve
```

### Compiles and minifies for production 编译和压缩到生产环境
```
npm run build
```

### Run your tests 运行单元测试
```
npm run test
```

### Lints and fixes files 整理和修复文件
```
npm run lint
```
